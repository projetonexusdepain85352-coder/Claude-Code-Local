# Area Inutil

Tudo que esta em `inutil/` foi mantido por rastreabilidade, nao por valor operacional direto.

## Subpastas

- `duplicatas_arquivadas/`: copias preservadas para comparacao, mas fora do fluxo normal.
- `metadados_de_origem/`: metadados e artefatos de arquivamento usados para provar origem e contexto.

## Regra pratica

Se o objetivo for estudar o runtime, comece em `../util/`.
Entre em `inutil/` apenas quando precisar auditar origem, duplicatas ou cadeia de custodia dos arquivos.
