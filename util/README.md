# Area Util

Tudo que esta em `util/` serve a pelo menos um destes papeis:

- codigo principal de estudo;
- referencia complementar;
- documentacao propria;
- preservacao de artefatos tecnicos relevantes.

## Subpastas

- `codigo_principal/`: base principal do runtime.
- `fontes_preservadas/`: bundles e compactados mantidos por integridade historica.
- `referencia_sanbuphy/`: fonte complementar, traducoes e analise comparativa.
- `documentacao/`: documentacao autoral deste acervo.

## Caminho recomendado

1. `documentacao/README.md`
2. `codigo_principal/claude_code_src/`
3. `referencia_sanbuphy/`
