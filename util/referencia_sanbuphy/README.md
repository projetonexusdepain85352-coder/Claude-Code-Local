# Referencia Sanbuphy

Esta area concentra o material complementar preservado do repositorio `sanbuphy/claude-code-source-code`.

## Subpastas

- `fonte_original/`: repositorio complementar preservado com `src/`, `docs/`, `scripts/`, `package.json` e `tsconfig.json`.
- `documentacao_traduzida_pt_br/`: traducao tecnica existente dos documentos principais.
- `analise_de_diferencas/`: comparativo entre a base complementar e o `codigo_principal`.

## Quando usar esta area

Use esta referencia quando precisar:

- consultar contexto de build;
- ler a documentacao original preservada;
- comparar `src/` complementar com a base principal;
- validar se determinado modulo existe so na fonte complementar.
